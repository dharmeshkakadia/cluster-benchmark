cluster-benchmark
=================

##How to use

1. Download it.
  ```
  git clone https://github.com/dharmeshkakadia/cluster-benchmark && cd cluster-benchmark
  ```

2. Compile it.
  ```
  mvn package
  ```
 
3. Run it.
  ```
  hadoop jar target/cluster-benchmark.jar [BenchmakrName]
  ```
